# FoodTherapy

#### 项目介绍
该项目是以养生食疗为主题的网站，具有养生知识推荐，养生菜谱搜索，测试身体状况，记录已做事情等功能

#### 技术栈
vue2 + vuex + vue-router + webpack + ES6 + axios + elementUI + 阿里图标iconfont
